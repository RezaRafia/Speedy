How It Works

Get up and running in just a few simple steps

1

Download & Install

Download Speedy from the releases page and install it on your Windows, Mac, or Linux computer.

2

Configure Load Percentages

Open the app and set the load percentage for each of your connections. The total must equal 100%. Set to 0% to disable a connection.

3

Power On & Enjoy

Click the power button to start the proxy server. Configure your system to use the SOCKS proxy (default port 8080) and enjoy faster speeds!

Technical Details

SOCKS5 Proxy Server

Speedy runs a local SOCKS5 proxy server that intelligently routes traffic across your connections.

Automatic Network Detection

Automatically detects all available network interfaces on your system.

Customizable Port

Configure the proxy server port to your preference (default is 8080).

Real-Time Logging

Monitor connection activity with built-in debug logging.

Simple & Intuitive Interface

Clean, modern design that makes connection management effortless

The interface shows all your network connections with easy-to-adjust load percentages, a simple power toggle, and real-time logging for transparency.
